Scenario fixes are packaged as ZIP files with the proper directory structure for use with Train Simulator Classic 2024.

To install a scenario fix ZIP:
1. Locate your Train Simulator Classic 2024 installation directory; it will be called "RailWorks." If installed through Steam, it is typically located at `C:\Program Files (x86)\Steam\steamapps\common\RailWorks\`.
2. Extract the contents of the ZIP file into that folder. The internal directory structure places the files in the correct directories.
3. Verify installation in Train Simulator 2024; the name will be updated to include a version numer.
